Linguagem (C)

A linguagem C é uma linguagem de programação criada em 1972 por Dennis Ritchie nos laboratórios da Bell Labs. Ela foi desenvolvida para criar o sistema operacional Unix e se tornou uma das linguagens mais importantes da computação. C é considerada uma linguagem de baixo nível, pois permite maior controle sobre o hardware e a memória do computador, tornando os programas muito rápidos e eficientes. Por isso, é amplamente utilizada no desenvolvimento de sistemas operacionais, drivers, sistemas embarcados, compiladores e softwares que exigem alto desempenho. Sua sintaxe serviu de base para diversas linguagens modernas, como C++, Java e C#. Apesar de ser mais difícil que linguagens de alto nível, aprender C ajuda a entender melhor o funcionamento interno dos computadores e os fundamentos da programação.

Principais tipos de Variaveis

Tipo:int

Oque faz:Armazena números inteiros (ex: 1, 10, -5)

Exemplo: int idade = 25;

Tipo:float

Oque faz:Armazena números com casas decimais (ex: 3.14)

Exemplo: float Salario = 1.518,50;

Tipo:double

Oque faz:Armazena números com casas decimais com maior precisão e capacidade que o float.(ex: π)

Exemplo: double pi = 3.1415926535;

Tipo:char

Oque faz:Armazena caracteres (ex: a,b,c...z,!,@ (pontuacoes no geral))

Exemplo: char Nome[50] = Ricardo Junior ;
